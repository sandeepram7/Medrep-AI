The canonical data lives in `data/*.json` files.

## Why these are incomplete

These were generated by Perplexity AI research prompts that covered a subset of drugs. The final JSON datasets in `data/` were built from these plus additional manual verification against official PDFs (CGHS MSO 2014, VMS 2023, ESIC dispensary schedule).

**Do not ingest these files into ChromaDB or any RAG pipeline.**
